Windows客户端

下载

下载并安装ShadowsocksR-win-4.9.0
要解压缩下载，请右键单击下载的文件，选择7-Zip选项集，然后将下载的文件解压缩到自己的文件夹中。新文件夹将具有名称ShadowsocksR-win-4.9.0。
在Windows资源管理器中，切换到ShadowsocksR的文件夹。
根据Windows的版本选择可执行文件：

  • 对于> = Windows 8或.NET 4.0，请使用ShadowsocksR-dotnet4.0.exe
  • 对于<= Windows 7或.NET 2.0，请使用ShadowsocksR-dotnet2.0.exe
    双击相应的应用程序文件以启动该程序。
添加服务器

要添加服务器，请打开系统托盘通知区域，然后找到ShadowsocksR图标。它看起来像纸飞机。右键单击ShadowsocksR图标以显示控件选项。选择服务器，然后编辑服务器…。

Android客户端

下载

下载并安装ShadowsocksR-android
点击下载的apk文件进行安装。它可能会被您的默认安全设置阻止。 按照提示转到“设置”。您需要选中该框以允许从未知来源 安装以允许安装。 确保选中该框以允许此安装，然后点击确定。

添加服务器

可以通过键入其详细信息或使用Android设备摄像头扫描显示的QR码来添加新服务器。

添加服务器后，点击右上角的连接按钮。
